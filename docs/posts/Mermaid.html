                                # 常用mermaid格式

                                # 甘特图

                                ```mermaid
gantt
  #标题
  title Test
  #时间格式
  dateFormat YYYY-MM-DD
  
  #创建任务A,使用`section`
  section A
  task1    :done,x1,2021-01-01,2020-01-06
  task2    :active,x2,2021-01-01,4d
  #格式: [任务名or描述  :状态,id,开始时间,结束时间(如后接持续时间则不用添加),持续时间
  
  #状态: done--已完成,active--进行中,crit--高亮
  task3    :active,x3,2021-01-03,5d
  task4    :crit,x4,2021-01-05,7d
  task5    :active,x5,2021-01-01,2021-01-12#补充,更好看
  
  #创建任务B
  section B
  task A   :active,xA,2021-01-02,2d
  task B   :active xB,after xA,1d
  #after (id),创建在某个任务后,如果使用此状态则`无法自定义开始时间`
  task C   :48h
```

                                # 常用日期格式:

                                格式例描述
                                YYYY2021四位年数
                                YY21两位年数
                                M MM1,2 ... 12月数
                                MM MMMM一月,二月,十二月月名
                                H HH0,1,2 ... 23二十四小时制
                                h hh1,2,3 ... 12十二小时制
                                a AAM, PM上午, 中午
                                m mm0,1,2 ... 59分钟
                                s ss0,1,2 ... 59秒
                                S(0.)0,1,2 ... 9分秒

                                # 流程图

                                # 样式 一

                                # 1. 方向

                                Code方向
                                TB从上到下
                                TD从上到下
                                BT从下到上
                                RL从右到左
                                LR从左到右

                                # 2. 各个形状

                                自定义id格式样式
                                1[ ]直角矩形
                                2( )圆角矩形
                                3([ ])体育场形
                                4[[ ]]长灯光形
                                5[( )]圆柱体形
                                6(( ))正圆形
                                7> ]标签形
                                8{ }菱形
                                9六角形
                                10[/ /]平行四边形
                                11[\ ]反向平行四边行
                                12[/ /]梯形
                                13[\ /]倒梯形

                                # 3. 箭头样式

                                样式格式&默认长度长度+1长度n
                                直线箭头-->--->
                                直线箭头内文字--内容-->--内容--->在内容后↓
                                直线-------添加所需↓
                                直线内文字--内容-----内容----符号即可
                                虚直线箭头-.->-..->
                                虚直线箭头内文字-.内容.->-.内容..->
                                虚直线-.--..-
                                虚直线内文字-.内容.--.内容..-
                                粗箭头==>===>
                                粗箭头内文字内容>内容=>注: 较长连线可↓
                                粗直线=======能需高版本支持
                                粗直线内文字内容=内容==

                                # 样式 二

                                所有可用样式与样式 一的相同, 额外格式:

                                样式格式
                                B端点--o/==o/-.-o
                                A,B端点o--o/o==o/o-.-o
                                B端x--x/==x/-.-x
                                A,B端xx--x/x==x/x-.-x
                                如下图& 注: 样式一也可以使用
                                PS:o/x可调换位置以上只是一点

                                # 多层嵌套

                                ```mermaid
graph TB
  z1 --> x2
  #利用subgraph创建出不同的块
  subgraph A
  x1 --> x2
  #利用end结尾
  end
  
  subgraph B
  y1 --> y2
  end
  subgraph C
  z1 --> z2
  end
```

                                # 时序图

                                # 0. 常用箭头

                                格式作用
                                ->>主动发出消息
                                -->>响应
                                -x异步消息

                                # 1. 消息

                                ```mermaid
sequenceDiagram
  #`participant xxx`声明参与则
  participant Client
  participant Server
  
  Client ->> Server : Get request.
  Server -x Client : Back data.
  Client -->> Server : Request fail.
  #格式: [参与者 箭头 参与者 : 文本]
```

                                # 2. 激活框

                                ```mermaid
sequenceDiagram
  participant Server
  participant Client
  
  #通过`+`,`-`来标记开始和结束
  Server ->> + Client : Service will be ended after 5min.
  Client ->> - Server : Got it, It'll be done after 5min.
  
  Client ->> + Server : User want to get vip access
  Server -->> - Client : User doesn't have access permission
```

                                # 3.注释(note)

                                ```mermaid
sequenceDiagram
  #利用`Note`添加注释,方向:left of/right of/over
  Note left of Server : High performance
  Note right of Server : Low price
  Note over Server : IPv6 Supported
```

                                # 4. 循环(loop)

                                ```mermaid
sequenceDiagram
  Client ->> + Server : Connecting
  Server -->> - Client : Back details
  
  #利用`loop`开启循环,loop后为loop的标题
  loop 5 times/sec
    Client ->> + Server : Send request
    Server -->> - Client : Back data
  #利用end结束loop
  end
```

                                # 5. 选择(alt)

                                ```mermaid
sequenceDiagram
  Client ->> Server : Get data
  Server -->> Client : Back data
  
  #利用`alt`
  alt Data size > 10M
    Client ->> Server : Disconnect
  else Data size < 10M
    Server ->> Client : Continue
  #利用end结束alt
  end
  
  Server-->> Client : Finish
```

                                # 6. 可选(opt)

                                ```mermaid
sequenceDiagram
  Client ->> + Server : Connect mirror(IPv4)
  
  #利用`opt`创建可选
  opt IPv6
    Client -->> Server : Connect mirror(IPv6)
  #利用end结束opt
  end
```

                                # 7. 并行(par)

                                ```mermaid
sequenceDiagram
  Client ->> GPU : Send raw video
  
  #利用`par`创建并行,par后为标题
  par 并行
    GPU ->> GPU : Decoding
    GPU ->> Client : send back
  #利用end结束par
  end
```
